１． 概要 ===================================================

 このEPS-convはいろいろな形式の画像ファイルをEPS(Encapsulated
PostScript)形式に変換するためのプログラムです。TeXで書かれた
文章に貼り付ける絵を作成することができます。

標準で対応している画像形式は

・BMP形式 : Windows標準の4,8,16,24,32bitのデータ
            RLE圧縮, OS/2形式も可
            （拡張子 BMP, DIB, RLE）

・JPEG形式 : JFIF標準のJPEGデータ
             グレースケール, プログレッシブ, CMYKも可
             （拡張子 JPG, JPEG, JPE）

・PNG形式 : Portable Network Graphics形式のデータ
            （拡張子 PNG）

・GIF形式 : GIF87a形式やGIF89a形式のデータ
            インターレースも可, アニメーションは第一コマ目のみ
            （拡張子 GIF）

・MAG形式 : MPSなどのMAG形式のデータ
            16色, 256色のどちらでも可
            （拡張子 MAG）

です。さらに、Susieプラグインによりその他の形式にも対応します。

ASCII85, RunLength, DCT, Flateなどの各種フィルターにより
データを圧縮することもできます。

JPEG形式に対してはjpeg2psのようなラッパーとしての動作も可能です。


２． 動作環境 ===============================================

Windows 95/98/Me/NT/2000/XPの日本語版で動作すると思われます。
（動作確認はWindows XPでおこなわれています。）


３． インストール＆アンインストール =========================

 インストールはダウンロードしたファイルを実行するだけです。
 アンインストールはコントロールパネルの「アプリケーションの
追加と削除」（XPでは「プログラムの追加と削除」）から行って
ください。


４． ファイルの内容 =========================================

 EPS-conv.exe    ----------     実行ファイル
 readme.txt      ----------     このファイル
 help.htm        ----------     ヘルプファイル
 Srexhsp.dll     ----------     さくらのHSP拡張プラグインDLL
 imgctl.hpi      ----------     画像操作用プラグイン
 epsconv.exe     ----------     コンソール版実行ファイル
 setup-ts.exe    ----------     インストーラ
 setup-ts.inf    ----------     インストール用設定ファイル


５． 使い方 =================================================

 実行ファイルEPS-conv.exeを起動してからメニューの「ヘルプ(H)」
−「ヘルプ(H)」を選択しヘルプを参照してください。
